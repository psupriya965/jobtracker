# Job Tracker Website

A responsive and interactive Job Tracker web application built using HTML, CSS, and JavaScript.

## Features

- Add job applications
- Search jobs
- Filter jobs
- Responsive design
- Interactive UI
- Dashboard cards
- Local storage support
- Dark mode

## Technologies Used

- HTML5
- CSS3
- JavaScript
